Detailed Description

This toys allows the grouping of several AnalogAlpha toys (e.g. AnalogAlphaToy or DirectOutput.Cab.Toys.Hardware.Lamp) into a matrix, which can be controlled by the matrix effects.

Note
Be sure to define this toy in the config file before the toys, which are listed in the ToyNames array.
